# Plan: Category Management Feature

## Objective
Add category management functionality to the Tasks section with a "General" default category created automatically for new users.

## Files to Create/Modify

### 1. Create Category Model
**File:** `lib/models/category.dart`
```dart
class CategoryModel {
  final String id;
  final String name;
  final String userId;
  final DateTime createdAt;

  CategoryModel({
    required this.id,
    required this.name,
    required this.userId,
    required this.createdAt,
  });

  Map<String, dynamic> toJson() => {
    'id': id,
    'name': name,
    'user_id': userId,
    'created_at': createdAt.toIso8601String(),
  };

  factory CategoryModel.fromJson(Map<String, dynamic> json) => CategoryModel(
    id: json['id']?.toString() ?? '',
    name: json['name']?.toString() ?? '',
    userId: json['user_id']?.toString() ?? '',
    createdAt: json['created_at'] != null 
      ? DateTime.parse(json['created_at']) 
      : DateTime.now(),
  );
}
```

### 2. Add Categories to LocalStorageService
**File:** `lib/core/services/local_storage_service.dart`
- Add constant: `static const String categoriesKey = 'local_categories';`
- Add methods:
  - `List<Map<String, dynamic>> get categories` - get all categories
  - `Future<void> addCategory(Map<String, dynamic> category)` - add new category
  - `Future<void> updateCategory(String id, Map<String, dynamic> category)` - update category
  - `Future<void> deleteCategory(String id)` - delete category
  - `List<Map<String, dynamic>> getCategoriesByUserId(String userId)` - filter by user

### 3. Create Categories Management Screen
**File:** `lib/features/tasks/screens/categories_screen.dart`
- AppBar with title "Categorías"

- Each item: name, delete button (except "General")
- FAB to add new category
- Dialog for add/edit with TextFormField
- Delete confirmation dialog
- "General" category cannot be deleted (shown as disabled)

### 4. Add Button to TasksListScreen
**File:** `lib/features/tasks/screens/tasks_list_screen.dart`
- Add IconButton in AppBar actions (after search icon, before filters)
- Icon: `Icons.category_rounded`
- Navigate to: `/tasks/categories`

### 5. Modify AuthNotifier for Default Category
**File:** `lib/core/providers/auth_provider.dart`
- In `signUp` method, after creating user and before setting state:
  - Create default "General" category with:
    - id: generated unique id

### 6. Add Route to App Router
**File:** `lib/core/router/app_router.dart`
- Add route: `/tasks/categories` → CategoriesScreen
- Place in the tasks routes section
