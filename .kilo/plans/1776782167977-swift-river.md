# Plan: Registro mínimo con 4 campos esenciales

## Decisión: Registro mínimo de 1-2 pasos ( Progressive Disclosure)

---

## Campos esenciales para registro:

### Paso 1 (RegisterScreen) ✅ - ÚNICO PASO REQUERIDO
- **email** (requerido - para login)
- **password** (requerido - seguridad)
- **name** (requerido - personalización)
- **role** (estudiante/docente - para rutas)

### Datos opcionales (añadir después en Settings/Perfil):
- university → puede inferirse o añadirse después
- career → se puede añadir después (en settings/perfil)
- semester → se puede añadir después
- bio → completamente opcional, añadir después
- subjects → se añadirán automáticamente cuando inscriba cursos

---

## Schema Supabase (listo para después):
```sql
create table public.users (
  id uuid not null default extensions.uuid_generate_v4 (),
  email character varying(255) not null,
  name text not null,
  created_at timestamp with time zone null default now(),
  updated_at timestamp with time zone null default now(),
  career character varying null,
  bio text null,
  role text null default 'student'::text,
  university text null,
  "avatarUrl" text null default ''::text,
  semester smallint null,
  constraint users_pkey primary key (id),
  constraint users_email_key unique (email),
  constraint users_role_check check ( ( role = any (array['student'::text, 'teacher'::text]) ) )
)
```
