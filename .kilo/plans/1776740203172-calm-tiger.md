# Plan: Rediseño del Home y Bottom Navigation

## Objetivo
Rediseñar la página de inicio (Home) y mejorar el Bottom Navigation al estilo de apps sociales (WhatsApp, Instagram, Facebook).

---

## Estado Actual

### Bottom Navigation Actual
- 6 tabs: Inicio, Tareas, Calendario, IA, Grupos, Configuracion
- Estilo oscuro con animaciones sutiles
- Badge de notificaciones en Tareas

### Home Actual
- AppBar con avatar, búsqueda y notificaciones
- Tarjeta de saludo con fecha y racha
- Tareas para hoy (lista horizontal)
- Próximos vencimientos
- Mis materias
- Tarjeta de IA con insight

---

## Mejoras Propuestas

### 1. Bottom Navigation (_estilo app social_)
**Cambios:**
- Fondo más integrado (menos contraste)
- Labels sempre visibles
- Animación de "pulse" al seleccionar
- Usar el color de marca (verde #00C853) como highlight
- Floating navigation style opcional

**Estructura propuesta:**
| Tab | Ícono | Label |
|----|-------|-------|
| 0 | home_rounded | Inicio |
| 1 | check_box_rounded | Tareas |
| 2 | calendar_today_rounded | Calendario |
| 3 | auto_awesome_rounded | IA |
| 4 | group_rounded | Grupos |

### 2. Home Dashboard (rediseño)

**Nuevas secciones propuestas:**
1. **Header con saludo y stats**
   - Saludo contextual + nombre
   - Mini stats: tareas completadas hoy, racha actual
   
2. **Quick Actions Row**
   - Botones rápidos:Nueva tarea, Nueva nota,紧急
   
3. **Sección "Para hoy"** (rediseñada)
   - Cards horizontales con swipe
   
4. **Racha diaria** (destacada)
   - Visualización tipo Duolingo
   
5. **Próximos vencimientos**
   - Lista vertical con prioridad visual

6. **Categorías/Materias**
   - Grid o lista horizontal

7. **IA Insights** (mejorada)
   - Card destacado con acciones sugeridas
